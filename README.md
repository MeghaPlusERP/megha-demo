# Megha · Merchandiser Console — Live Preview

AI-first, jewellery-native ERP demo · merchandiser-facing daily workspace.

**Live URL:** [meghapluserp.github.io/megha-demo](https://meghapluserp.github.io/megha-demo/)

## What this is

A single-page HTML preview of the **Megha Retail** merchandiser console — built around 11 named AI agents that propose specific actions and execute after approval.

8 workspaces are wired:

- **Morning Brief** — today's actions across all agents
- **Stores · Network View** — bird's-eye over all stores with matrix view + per-store slide-over
- **Allocation** — pending decisions + 10-store inventory health grid
- **Discount & Waiver** — making-charge waiver pool, discipline by category, audit trail
- **Melt & Remake** — jewellery-only slow-mover cycle (the differentiator)
- **Schemes & Plans** — Gold / Diamond / Swarn Plus customer schemes
- **Sales & Margin · P&L** — investor-deck metrics laddering up from operations
- **Audit Trail · Board Pack** — listed-entity governance (IFC, CARO, SEBI LODR, CSCRF)

Plus a Claude-style **Ask Megha** chat with pre-seeded management Q&A.

## Notes

- All numbers, customer names, store-level financials, and audit entries are **illustrative** — invented for demo purposes.
- Light/dark theme toggle in top bar (☀/☾).
- Cmd+K palette for navigation.
- Click any store, action row, or chat thread to drill in.
